# Changelog

Todas as mudanças notáveis neste projeto serão documentadas neste arquivo.

O formato é baseado em [Keep a Changelog](https://keepachangelog.com/pt-BR/1.0.0/),
e este projeto adere ao [Versionamento Semântico](https://semver.org/pt-BR/).

## [1.0.0] - 2026-03-29

### Adiciona
- ✨ Autenticação de usuários com JWT
- ✨ Criação e edição de perfil do corredor
- ✨ Geração automática de planos de treino periodizados
- ✨ Suporte a 4 fases: Base, Desenvolvimento, Pico e Taper
- ✨ Registro de execução de sessões de treino
- ✨ Dashboard com visualização de progresso com gráficos
- ✨ Interface responsive React com React Router
- ✨ API REST completa com Spring Boot 3
- ✨ Banco de dados PostgreSQL com Hibernate JPA
- ✨ CORS configurado para frontend
- ✨ Testes automatizados (Unit + Integration)
- ✨ Análise estática com Checkstyle e SpotBugs
- ✨ ESLint e Prettier para frontend
- ✨ GitHub Actions CI/CD pipeline
- ✨ Documentação completa com README.md
- ✨ Versionamento semântico implementado

### Melhorias
- 🔒 Spring Security com criptografia de senhas
- 🔒 Validação de entrada de dados
- 🔒 Proteção CORS habilitada
- 📊 Testes com cobertura de casos principais
- 📊 Padrão de código consistente

### Correções
- 🐛 Tratamento de erros em autenticação
- 🐛 Validação de data em plano de treino

### Dependências
- Java 21 LTS
- Spring Boot 3.4.3
- React 19.2.4
- PostgreSQL 15+
- Maven 3.8+
- Node.js 18+

---

**Nota:** Esta é a versão inicial da aplicação, desenvolvida como projeto de conclusão de Engenharia de Software.
